跨平台适配方案（视频点播）：

跨平台方案X1

（基于普通HTTP协议）

  • 实例演示 X1-1
  • PC端Http + Flash + Mp4
  • 安卓端,苹果端,WPHttp + Html5 + Mp4
方案X1说明：本实例基于HTTP协议，是应用最广泛的实例。
1.基于HTTP协议(比如IIS, Apache , Nginx 等网站环境下的http协议都可以正常使用本方案)
2.注意:视频文件必须是标准MP4视频(H264+aac)
3.注意:确认服务器是否支持MP4视频的mime类型，参考链接>>

跨平台方案X2

（基于普通HTTP协议+ RTMP协议）

  • 实例演示 X2-1
  • PC端RTMP + Flash + Mp4
  • 安卓端,苹果端,WPHttp + Html5 + Mp4
方案X1说明：本实例基于HTTP协议+RTMP协议。
1.本实例要求服务器端必须有装 SRS, crtmpserver, FMS , RED5 等流媒体软件做支持
2.注意:视频文件必须是标准MP4视频(H264+aac)

跨平台方案X3

（基于普通HTTP协议+ RTMP协议）

  • 实例演示 X3-1
  • PC端,安卓端RTMP + Flash + Mp4
  • 苹果端,WPHttp + Html5 + Mp4

方案X3说明：本实例基于HTTP协议+RTMP协议。
1.本实例要求服务器端必须有装 SRS, crtmpserver, FMS , RED5 等流媒体软件做支持
2.注意:视频文件必须是标准MP4视频(H264+aac)

3.注意: 安卓端，请先安装flash插件

跨平台方案X4

（基于HTTP协议 + RTMP协议 + HLS）

  • 实例演示 X4-1
  • PC端RTMP + Flash + Mp4
  • 安卓端,WPHttp + Html5 + Mp4
  • 苹果端Http + Html5 + m3u8
方案X4说明：本实例基于HTTP协议+RTMP协议+HLS。
1.本实例要求服务器端必须有装 SRS, crtmpserver, FMS , RED5 等流媒体软件做支持
2.注意:视频文件必须是标准MP4视频(H264+aac)
